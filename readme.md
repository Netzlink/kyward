# Kyward
A simple token management tool written in almost completly rust.

![Doors Section Screenshot](assets/doors_section.png)

## Setup 
_Dev_
```bash
make run
```
_Docker_
```bash
docker build -f Dockerfile . -t kyward
docker run --rm -p 8000:8000 kyward:latest
```
